Anyone can Code

It's time to move beyond text and inlcude images and videos! This is actually not very hard at all. For an image, all you need to do is use the src (short for source) attribute of the img (short for image) tag like this: <img src="https://somesite.com/image.png"/>. You can also specify height, width, a frameborder, and some text for if the image fails to load like this: height="100". This is another one where you should just put a slash before the closing angle bracket. Also, it doesn't have to be a png. The img tag supports most image formats. Videos are very similar to do, and the way you embed a video is the same way you can actually embed someone else's site into your own. For this you can use the <iframe> and </iframe> tags. Again you will need to use the src attribute, and you can still specify the size of the iframe using height and width attributes. Take a look at this example:
<!DOCTYPE html>
<html>
<head>

</head>
<body>
  <img src="https://d4q2.github.io/SyntaxHighlightCodeEditor/Files/Code.jpg" alt="Monitor displaying code" height="100" width="200" frameborder="1"/>
  <iframe width="200" height="100" src="https://www.youtube.com/embed/tgbNymZ7vqY"></iframe>
</body>
</html>
