Anyone can Code

So you want to be able to change pages on your website or let the user navigate to another site? Easy. Simply use the anchor tag pair: <a> and </a>. However, in this case we are going to need to tell the computer that is looking at our website where to go like this: <a href="https://www.mysite.com"></a>. And, just like lots of the other tags, we'll need to put some text in between the anchor tags so that our users can see the link. Check out the example below:
<!DOCTYPE html>
<html>
<head>

</head>
<body>
  <!-- Note: This is not currently working in the code editor. -->   <a href="https://www.google.com">Google Link</a>
</body>
</html>
