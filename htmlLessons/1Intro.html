Anyone can Code

HTML stands for Hyper Text Markup Language. It is the base language used for writing on the web. You can combine CSS (Cascading Style Sheets) to make it look better and add JavaScript to make it function, but it will all have its foundations in HTML. HTML is mostly made up of pairs of tags that have a start tag like this: <something> and an end tag like this: </something>. To start an HTML file, put <!DOCTYPE html>. This does not require an end tag and we don't need to worry about what it does right now, just remember to put it at the top of your file. Everything else on your webpage must be surrounded by <html> and </html>. So put the start tag (<html>) on the line after the <!DOCTYPE html> and the end tag (</html>) on the line after that. An example is below:
<!DOCTYPE html>
<html>
  Your content will go here!
</html>
