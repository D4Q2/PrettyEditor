Now lets learn about a great feature in html. The input tag. This is another one that really needs javascript to unlock its full potential, but you'll get there soon enough. The input tag can be written as <input/>, and when you do that you do not need an end tag. To use it, you will need to assign it a type to it, and in some cases you can also assign it a placeholder and/or a value. Check it out here:
<!DOCTYPE html>
<html>
<head>

</head>
<body>
  <input type="text" placeholder="Username">
  <input type="password" placeholder="Password">
  <input type="color" value="rgb(0, 0, 0)">
<!-- There are also many more types, these are just a few!-->
</body>
</html>
