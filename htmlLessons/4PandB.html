Now we'll learn about some basic formatting elements. First up are <p> and </p>, the p being short for paragraph. Adding these around some text will make it a separate paragraph. Another tag you can use to break up lines is the <br/> tag, which is short for break. It has no end tag. Often in the source code of a website the programmers will start a new line after a <br/> tag, just to help it look a little more like it really will, but this is not necessary. You can see an example of these below:
<!DOCTYPE html>
<html>
<head>

</head>
<body>
  This will be on the same line
  as this, which is why we need the br/ tag.
  <p>This is my first paragraph.</p>
  <p>This is my second paragraph.</p>
  Words on this line<br/> these will go on the next line.
  <br/>   This setup<br/>
  makes it easier to tell these will be on a new line.
</body>
</html>
