That's all of the main html tags save two, which I am about to show you. It may not seem like it, but you have already made huge headway into the world of web development. You can already assemble a basic website. Now you just need some CSS (to make it look nicer) and JavaScript (to add more movement and functionality) to make your own high-quality website. To use CSS in a webpage you need to put it (usually in the head) inside of <style> and </style>. To use JavaScript, you need to put it between <script> and </script> tags in either the head or the body.
<!DOCTYPE html>
<html>
<head>

  </style>
  <script>
    
  </script>
</head>
<body>

</html>
