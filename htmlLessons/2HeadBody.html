Anyone can Code

Now we are going to learn about the most basic and commonly used tags in HTML (after <html></html>). First are the <head> and </head> tags. These tags will be placed inside the HTML tags. Typically programmers will use the head area to define things about how the page will look and act. The <body> and </body> tags (outside of and after the <head></head> tags, but inside of the <html></html> tags) will then actually contain the page content. Look at this example:
<!DOCTYPE html>
<html>
<head>

</head>
<body>
  Don't worry, we'll make this look more interesting soon.
</body>
</html>
