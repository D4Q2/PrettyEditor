CSS stands for Cascading Style Sheets. It is made to add, well, style. Colors, fonts, sizes, and much more. In this lesson, we will use change the text alignment of some text. There are three places to write css. The first is to put it right in with the tag it is going to affect, like this: <div style="text-align: center;"></div>. Your css would go in the style attribute of a tag. This does not have to be a div tag, it works with almost all tags. So do the other methods. Another method is to put it in another file, but it while this is generally the best method long-term, the last method is good for prototyping and what we are doing. The last method, which is to put the css information inside of style tags: <style></style>, and then connect use classes to make it work with our content. For this, we can either connect it to a specific tag such as body or div like this <style> body { text-align: center; } </style> or we or we can use the class attribute of an html tag like this: <style> .area { text-align: center; } </style> and later on we can do: <div class="area"></div> or <p class="area"></p>. Remember, we put the dot in front of the name when we use the class attribute in the tags, and we don't when we are using a tag. Have a look at this example:
<!DOCTYPE html>
<html>
  <head>
    <style>
      h1 {
        text-align: center;
      }
      .main {
        text-align: left;
      }
    </style>
  </head>
  <body>
    <h1>Title</h1>
    <div class="main">
      Lots of words and words and more words.
    </div>
  </body>
</html>
