Anyone can Code

Another important thing to learn is how to make comments. Comments, in programming, are pieces of information that are in the source code but will not be included in the output. They are often used to annotate what certain parts of the code are doing, or why certain lines have been included. You can make a comment by typing <!-- before it and --> after it. Everything in between those symbols will not be shown on your webpage. Try running this code:
<!DOCTYPE html>
<html>
<head>

</head>
<body>
  This can be seen.
  <!-- But this cannot -->
</body>
</html>
